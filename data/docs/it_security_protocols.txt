PrivAI IT Security Protocols

1. Password Management
Passwords must be at least 12 characters long, containing uppercase, lowercase, numbers, and symbols. Passwords must be changed every 90 days.

2. Access Control
Access to internal systems is granted on a least-privilege basis. Multi-Factor Authentication (MFA) is mandatory for all internal and external services.

3. Data Encryption
All sensitive data at rest and in transit must be encrypted using AES-256 or equivalent standards.

4. Device Security
Company laptops must have disk encryption and up-to-date antivirus software. Public Wi-Fi is prohibited unless a company-approved VPN is used.

5. Incident Reporting
Any security breach or suspicious activity must be reported to the IT Security team immediately via the "SecureReport" portal.
